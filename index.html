OPTIONS

jameson interprets the following options:

  • -b The Bash interpreter is enabled (Optional dependencies: AWK, SED, grep).
  • -c [file] Data will be formatted as defined by the included CSS stylesheet.
  • -h [file] Data is structured according to the included HTML template.
  • -j JavaScript interpreter enabled.
  • -p PHP interpreter is enabled.
  • -y Python interpreter is enabled.

  • --chef (experimental) Takes a list of ingredients, and outputs a completed dish.

  • --cycle Enables organic human transport protocol.

  • --game Allocate resources to video game proficiency.

  • --photo Capture and process images, as keepsakes and entertainment.

  • --write Will process and generate literature.

  • --hipster ** Deprecated ** Did things before they were cool, before it was cool.
